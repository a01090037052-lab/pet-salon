## 1. 소개

**펫살롱**은 애견 미용 샵을 위한 고객 관리 프로그램입니다. 고객과 반려견 정보 관리, 예약 관리, 미용 기록, 매출 추적을 한 곳에서 할 수 있습니다.

| 할 수 있는 것 | 앱의 특징 |
|---|---|
| 고객 + 반려견 정보 관리 | 완전 무료, 구독료 없음 |
| 예약 관리 (월간/타임테이블) | 오프라인 사용 가능 |
| 미용 기록 + 사진 | 데이터가 내 기기에 저장 |
| 실시간 매출 추적 | GitHub Pages로 어디서든 접속 |
| 일별/주간/월간 매출 시각화 | JSON 파일로 백업/복원 |
| 미용사별 성과 분석 | 예약 알림 (브라우저 알림) |
| 세무 자료 CSV 내보내기 | 저장소 용량 모니터링 |

## 4. 초기 설정

### 1단계: 매장 정보 입력

**더보기 > 설정 > 매장관리 탭**

| 항목 | 예시 |
|---|---|
| 매장 이름 | 해피펫살롱 |
| 연락처 | 02-1234-5678 |
| 주소 | 서울시 강남구 ... |
| 재방문 알림 기준 | 30일 (기본값) |

### 2단계: 미용사 등록

**설정 > 매장관리 탭 > 미용사 관리**에서 미용사 이름 추가

### 3단계: 휴무일 설정

**설정 > 매장관리 탭 > 영업설정**에서 정기 휴무일 표시

### 4단계: 알림 설정

**설정 > 운영설정 탭**

1. **예약 알림 활성화** 토글 켜기
2. **알림 시간** 선택 (10분/30분/1시간/2시간 전)

### 5단계: 서비스 메뉴 등록

**더보기 > 서비스 메뉴** (필요시 수동 추가)

- 소형/중형/대형 3단계 가격 설정
- 활성/비활성 토글로 사용 여부 관리

### 6단계: 첫 고객 + 반려견 등록

1. **고객 관리** > **+ 새 고객**
2. 이름, 연락처 입력 후 저장
3. 고객 상세에서 **+ 반려견 등록**
4. 반려견 이름, 견종, 나이, 사진 입력

## 5. 일일 사용 가이드

### 아침 - 하루 시작

1. **대시보드** 확인
   - 백업 알림이 있으면 "백업하기" 클릭
   - 오늘 예약 목록 확인
   - 일일 매출 목표 진행률 확인
   - 재방문 알림 > 연락할 고객 확인
   - 생일 알림 > 축하 문자 버튼
   - 미수금 현황 > 결제 처리 버튼

### 전화 예약 받을 때

1. **예약 관리** > **+ 새 예약**
2. 고객 이름 또는 전화번호로 **검색** (검색형)
3. 반려견 선택 (1마리면 자동 선택)
4. 날짜 + 시간 입력
5. **선택사항**: 서비스 선택
6. **상세 옵션**: 미용사, 예상 소요시간, 메모 (필요시)
7. **저장** > 예약 상태 "대기"

> 예약 관리에서 **캘린더 보기** 또는 **타임테이블 보기** 버튼으로 한눈에 확인 가능. 시간 범위 충돌 자동 감지.

### 미용 완료 후

1. 예약 상태를 **"완료"**로 변경
2. **미용 기록** > **+ 새 기록** 또는 대시보드에서 **기록** 버튼

| 기능 | 설명 |
|---|---|
| 고객 검색 | 이름/전화번호로 즉시 검색 |
| 반려견 사이즈 감지 | 몸무게 기반 소형/중형/대형 자동 설정 |
| 가격 자동 계산 | 서비스 체크 시 사이즈에 맞는 가격 합산 |
| 최종 금액 입력 | 할인/추가요금 반영 |
| 결제 수단 | 현금/카드/이체 |

### 마감 - 매출 확인

**매출 페이지** (하단 네비)에서:

- **일일 매출 목표** 달성률 확인
- **오늘/주간/월간 매출** 차트 확인
- **미용사별 성과** (건수/매출) 분석
- **월별 아카이브**에서 과거 매출 확인
- **세무 자료 CSV 내보내기**

## 6. 메뉴별 상세 기능

### 대시보드 (홈)

| 영역 | 내용 |
|---|---|
| **인사말 배너** | 시간대별 인사 + 매장명 + 오늘 요약 |
| **백업 알림** | 7일 이상 백업 안 했으면 경고 |
| **통계 카드** | 오늘 예약 / 이번 달 매출 / 고객 수 / 오늘 매출 |
| **일일 매출 목표** | 목표액 + 프로그레스 바 + 달성률 |
| **오늘 예약 목록** | 상태(대기/미용중/완료) + 기록 버튼 |
| **재방문 알림** | 기준일 초과 고객 + 문자/전화 버튼 |
| **생일 알림** | 고객/반려견 생일 축하 문자 버튼 |
| **미수금 현황** | 외상 잔액 표시 + 결제 처리 버튼 |

### 고객 관리

**목록 기능**:
- **검색**: 이름/전화번호 즉시 검색
- **정렬**: 이름순, 최근방문순, 등록일순, 방문횟수순
- **자동 분류**: 신규(1-3회) / 일반(4-10회) / 단골(11회+)
- **장기 미방문 경고**: 30일+/60일+ 배지 표시
- **20명씩 로드** + "더 보기" 버튼

**고객 상세 페이지**:
- **기본 정보**: 이름, 연락처, 주소, 생일
- **방문 통계**: 총 방문 횟수, 총 지출, 평균 객단가
- **미수금 잔액**: 현재 외상액
- **반려견 목록**: 고객의 모든 반려견
- **미용 기록**: 고객의 전체 미용 기록 조회

### 반려견 관리

**등록/수정**:
- **나이 직접 입력**: 나이 입력 시 자동으로 생년월일 계산, 매년 자동 업데이트
- **사진**: 프로필 사진만 DB 저장 (자동 압축)
- **기본 정보**: 반려견 이름, 견종, 나이, 성별, 중성화 여부, 몸무게
- **사이즈**: 소형/중형/대형 자동 판정 또는 수동 설정
- **특이사항**: 성격/행동, 건강, 알러지, 메모

### 예약 관리

| 기능 | 설명 |
|---|---|
| **월간 캘린더 뷰** | 날짜별 예약 건수 색상 코딩 |
| **미용사별 타임테이블 뷰** | 시간대 x 미용사 그리드 |
| **빠른 필터** | 전체/오늘/내일/이번주 |
| **상태 필터** | 대기/미용중/완료/취소/노쇼 |
| **검색형 고객 선택** | 이름/전화번호로 실시간 검색 |
| **예상 소요시간** | 30분~3시간 범위 설정 |
| **시간 범위 충돌 감지** | 같은 시간 같은 미용사/반려견 중복 경고 |

**예약 상태**:

| 상태 | 의미 |
|---|---|
| 대기 | 아직 확정 안 됨 |
| 미용중 | 현재 미용 진행 중 |
| 완료 | 미용 완료 |
| 취소 | 고객 취소 |
| 노쇼 | 예약 시간에 안 옴 |

### 미용 기록

| 기능 | 설명 |
|---|---|
| **검색형 고객 선택** | 이름/전화번호 실시간 검색 |
| **필수 항목 우선** | 고객/반려견/날짜/미용사 필드 상단 |
| **선택사항** | 서비스 선택 (필수 아님) |
| **상세 옵션** | 추가 정보 토글로 접기/펼치기 |
| **할인/추가요금** | 입력 시 최종금액 자동 계산 |
| **결제 수단** | 현금/카드/이체 |

### 매출 페이지

**별도 분리된 네비 메뉴** (하단 네비 5번째)

| 기능 | 설명 |
|---|---|
| **일일 매출 목표** | 목표액 설정 + 프로그레스 바 |
| **오늘/주간/월간 매출 차트** | 시각화 + 상세 데이터 |
| **미용사별 성과 대시보드** | 건수/매출/평균단가 |
| **월별 아카이브** | 지난 달 매출 비교 |
| **세무 자료 내보내기** | CSV 다운로드 |

### 설정 (3개 탭)

#### 탭 1: 매장관리
- **매장 정보**: 매장명, 연락처, 주소, 재방문 알림 기준일
- **미용사 관리**: 미용사 추가/수정/삭제
- **영업설정**: 정기 휴무일 표시

#### 탭 2: 운영설정
- **알림 설정**: 활성화 토글 + 알림 시간 선택
- **매출 목표**: 일일 매출 목표 설정
- **고정비**: 월 고정비 (만원 단위)
- **메시지 템플릿**: 재방문/예약/생일/완료 문자 템플릿
- **테마 색상**: 8가지 색상 선택

#### 탭 3: 데이터
- **데이터 현황**: 고객/반려견/예약/기록/서비스 건수
- **세무 자료 내보내기**: CSV 다운로드
- **백업 & 복현**: JSON 내보내기/가져오기
- **휴지통**: 삭제된 항목 복원 또는 완전 삭제
- **전체 데이터 초기화**: 모든 데이터 삭제 (2중 확인)

### 하단 네비게이션 (6개 메뉴)

```
홈 | 고객 | + | 예약 | 매출 | 더보기
```

- **홈**: 대시보드
- **고객**: 고객 관리
- **+**: 빠른 메뉴 (새 고객/반려견/예약/기록 추가)
- **예약**: 예약 관리
- **매출**: 매출 분석 (별도 페이지)
- **더보기**: 반려견 관리, 서비스 메뉴, 설정

## 7. 다른 사람이 처음 사용할 때

### 다른 매장에 공유하기 전에 알아두세요

**데이터는 절대 섞이지 않습니다!**
- 같은 URL을 사용해도 각자 기기의 브라우저에 독립적인 데이터 저장
- 각 매장이 독립적으로 자신의 데이터만 관리 가능

### 상대방에게 공유하는 방법

1. **앱 URL 공유**: `https://a01090037052-lab.github.io/pet-salon/`
2. **상대방이 접속하면 바로 사용 가능**
3. **각자 초기 설정만 하면 끝**

### 상대방의 첫 사용 가이드

#### Step 1: 앱 접속 (2분)
1. 브라우저에서 URL 접속
2. 홈 화면에 추가 (필수)
3. 홈 화면의 앱 아이콘 탭

#### Step 2: 필수 초기 설정 (10분)
**더보기 > 설정 이동**

1. **매장관리 탭**:
   - 매장 이름 입력
   - 연락처, 주소 입력
   - 미용사 3명 정도 등록

2. **운영설정 탭**:
   - "예약 알림" 토글 ON

#### Step 3: 서비스 메뉴 등록 (선택)
**더보기 > 서비스 메뉴** > 필요한 서비스 추가

#### Step 4: 첫 데이터 입력 (5분)
1. **고객** 탭 > **+ 새 고객** > 저장
2. 고객 탭 > **+ 반려견 추가** > 저장

#### Step 5: 첫 예약/기록 (5분)
1. **예약** 탭 > **+ 새 예약** > 저장
2. **미용 기록** 탭에서 기록 입력

## 8. 데이터 관리 & 백업

### 백업 방법

1. **더보기 > 설정 > 데이터 탭 > "데이터 백업하기"**
2. JSON 파일 자동 다운로드
3. iCloud / Google Drive / 카카오톡에 보관

| 백업 주기 | 권장도 |
|---|---|
| 매일 | 가장 안전 |
| 주 2회 | 권장 |
| 주 1회 | 최소 |

> 7일 이상 백업하지 않으면 대시보드에 경고 표시

### 복원 방법

1. **더보기 > 설정 > 데이터 탭 > "데이터 복원하기"**
2. 백업 JSON 파일 선택
3. 확인 (현재 데이터가 백업 내용으로 교체됨)

### 폰 교체 시 데이터 이전

```
현재 폰:  설정 → "데이터 백업하기" → JSON 파일 저장
             ↓
        카카오톡, 에어드롭, 이메일로 전송
             ↓
새 폰:    같은 주소 접속 → 설정 → "데이터 복원하기" → 파일 선택
```

## 11. 문제 해결

### 데이터가 사라졌어요

1. 백업 파일이 있으면 → **설정 > 데이터** 탭 > **데이터 복원하기**
2. 없으면 수동 재입력 필요 (**정기 백업의 중요성!**)

### 업그레이드 후 화면이 이상해요

- 브라우저 캐시 문제. **강제 새로고침** (Ctrl+Shift+R 또는 캐시 삭제 후 접속)
- 그래도 안 되면 `git revert HEAD && git push`로 이전 버전 복구

### 알림이 안 와요

| 확인 | 방법 |
|---|---|
| 알림 활성화 | 설정 > 운영설정 탭 > 예약 알림 토글 켜기 |
| 브라우저 권한 | 브라우저 설정에서 알림 허용 확인 |
| 앱 열려있어야 함 | 브라우저/PWA가 열려있을 때만 작동 |

### 사진이 업로드 안 돼요

- 파일 크기가 너무 크면 느릴 수 있음
- 지원 형식: JPG, PNG, GIF, WebP
- 저장소 용량 경고 확인 (사진 압축 권장)
