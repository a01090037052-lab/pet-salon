📅 일일 사용법

🌅 아침 - 하루 시작

  1. 대시보드 탭 > 오늘 예약 확인
  2. 백업 알림이 있으면 "백업하기" 클릭
  3. 일일 매출 목표 프로그레스 바 확인
  4. 재방문 알림 고객에게 "문자" 또는 "전화" 버튼
  5. 생일 알림 있으면 축하 문자 전송

📞 전화 예약 받을 때

1
예약 > + 새 예약
2
고객 이름/전화번호로 검색 선택
3
날짜 + 시간 입력
4
선택사항 (서비스/미용사/메모)
5
저장

✂ 미용 시작→완료

Step 1: 예약 상태를 "완료"로 변경

Step 2: 대시보드 또는 예약 목록에서 "기록" 버튼 클릭

Step 3: 미용 기록 정보 입력 후 저장

💰 마감 - 매출 확인

매출 페이지 (하단 네비)에서:

  • 일일 매출 목표 달성률 확인
  • 오늘/주간/월간 매출 차트 확인
  • 미용사별 성과 분석
⭐ 주요 기능

고객 관리

  • 검색/정렬: 이름, 최근방문, 등록일, 방문횟수
  • 자동 분류: 신규(1-3회) / 일반(4-10회) / 단골(11회+)
  • 장기 미방문 경고: 30일+, 60일+ 배지
  • 20명씩 로드 + "더 보기" 버튼

반려견 관리

  • 나이 직접 입력: 자동으로 생년월일 계산, 매년 자동 업데이트
  • 프로필 사진: DB 저장 (자동 압축)
  • 상세 정보: 견종, 성별, 중성화, 몸무게 등

예약 관리

  • 캘린더 뷰: 날짜별 예약 건수 시각화
  • 타임테이블 뷰: 시간대 x 미용사 그리드
  • 시간 충돌 감지: 자동 경고
  • 검색형 고객 선택: 이름/전화번호 실시간 검색

미용 기록

  • 검색형 고객 선택: 빠른 선택
  • 가격 자동 계산: 사이즈별 가격 합산
  • 결제 수단: 현금/카드/이체

매출 페이지

  • 일일 매출 목표: 목표액 + 프로그레스 바
  • 차트 시각화: 오늘/주간/월간/6개월
  • 미용사별 성과: 건수/매출/평균단가
  • 세무 자료: CSV 내보내기

🔧 문제 해결

데이터가 사라졌어요

백업 파일이 있으면 → 설정 > 데이터 탭 > 복원하기. 없으면 복구 불가.

화면이 이상해요 (업그레이드 후)

강제 새로고침 (아래로 당기기 + 새로고침) 또는 캐시 삭제 후 접속.

알림이 안 와요

확인방법
알림 활성화설정 > 운영설정 > 예약 알림 토글 ON
브라우저 권한브라우저 설정에서 알림 허용 확인
앱 열림앱이 열려있어야만 작동

사진 업로드가 안 돼요

JPG, PNG, GIF, WebP를 지원합니다. 저장소 용량 경고도 확인하세요.

시간 충돌 경고가 자꾸 떠요

예상 소요시간을 확인하세요. 예: 14:00 시작, 1시간 → 14:00~15:00 점유.

❓ FAQ

홈 화면에 추가는 꼭 해야 하나요?

네. 앱처럼 사용 가능하고, Safari 데이터 자동 삭제를 방지합니다.

업그레이드하면 데이터가 날아가나요?

아니요. 코드만 업데이트되고 데이터는 그대로 유지됩니다.

다른 사람도 쓸 수 있나요?

네. URL만 공유하면 됩니다. 데이터는 각 기기에 별도 저장됩니다.

알림이 앱을 닫아도 오나요?

아니요. 앱이 열려있을 때만 작동합니다.

PC와 모바일 데이터가 달라요

정상입니다. 각 기기가 독립 데이터입니다. 백업/복원으로 동기화하세요.

반려견의 나이 입력 후 자동 업데이트되나요?

네. 나이를 입력하면 생년월일이 자동 계산되고, 매년 자동으로 업데이트됩니다.

고객이 신규/일반/단골로 자동 분류되나요?

네. 신규(1-3회), 일반(4-10회), 단골(11회 이상)으로 자동 분류됩니다.

고객 목록에서 20명씩만 보여요

맞습니다. 성능을 위해 20명씩 로드되며, 아래 "더 보기" 버튼으로 추가 로드됩니다.

미용 기록에서 사진을 못 본다고 했는데?

미용 기록 시 전/후 사진 기능이 제거되었습니다. 반려견 프로필 사진만 저장됩니다.

매출 페이지가 별도로 있나요?

네. 하단 네비 5번째 메뉴가 매출 페이지입니다. 일일 목표, 차트, 미용사 성과를 확인할 수 있습니다.

비용이 드나요?

완전 무료입니다. 지금도, 앞으로도.
