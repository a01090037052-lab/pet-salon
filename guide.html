📅 월간 캘린더 뷰 NEW

예약 관리에서 "캘린더 보기" 버튼을 탭하면 한 달 전체를 한눈에 볼 수 있습니다.

색상의미
파랑1~2건 예약
빨강3건 이상 예약
회색예약 없음

날짜를 탭하면 해당 날짜 예약만 필터링됩니다.

📷 반려견 사진 NEW

반려견 등록/수정에서 사진을 업로드할 수 있습니다. 카메라로 바로 촬영도 가능합니다.

  • 반려견 목록에서 사진 표시
  • 반려견 상세에서 큰 사진 표시
  • 사진 변경/삭제 가능

📊 매출 시각화 NEW

미용 기록 페이지에서 매출 카드를 탭하면:

  • 오늘 매출 → 건별 상세 내역
  • 이번 주 매출 → 일별 바 차트 (월~일)
  • 이번 달 매출 → 일별 바 차트 (1일~말일)

🔍 검색형 고객 선택 NEW

예약/미용기록 등록 시 고객을 이름이나 전화번호로 바로 검색할 수 있습니다. 기존 드롭다운 대신 검색 입력창이 표시됩니다.

🔔 예약 알림 NEW

설정에서 활성화하면 예약 시간 전에 브라우저 알림이 옵니다.

설정옵션
알림 시간10분 / 30분 / 1시간 / 2시간 전
조건앱이 열려있을 때만 작동

📂 월별 아카이브 NEW

미용 기록 페이지 하단에서 월별 매출 요약을 확인할 수 있습니다. 각 월을 펼치면 상세 내역을 볼 수 있습니다.

✂ 예약 시 미용 기록 자동 등록 NEW

새 예약 등록 시 "미용 기록도 함께 등록"을 체크하면, 예약 저장 후 미용 기록 폼이 자동으로 열립니다.

📈 주간 매출 차트 NEW

대시보드에 이번 주 매출이 일별 막대 차트로 표시됩니다. 오늘은 초록색으로 강조됩니다.

💾 백업 & 데이터 관리
🚨
백업은 필수입니다! 데이터는 한 번 잃으면 복구 불가능합니다.

백업 방법

1
설정 > 데이터 백업하기 클릭
2
JSON 파일 다운로드됨
3
iCloud / Google Drive / 카카오톡에 보관
주기권장도
매일⭐⭐⭐ 가장 안전
주 2회⭐⭐ 권장
주 1회⭐ 최소

복원 방법

설정 > 데이터 복원하기 > 백업 JSON 파일 선택 > 확인

폰 교체 시

현재 폰: 설정 > 백업 > 파일 저장 ↓ 카카오톡/에어드롭으로 전송 새 폰: 접속 > 설정 > 복원 > 파일 선택

🔄 업그레이드

PC에서 코드를 수정한 후:

git add index.html css/ js/ sw.js manifest.json git commit -m "기능 추가: 설명" git push

20~30초 후 자동 배포. 모바일에서 새로고침하면 적용됩니다.

✔
업그레이드해도 데이터는 안전합니다.
코드만 업데이트되고, 고객/예약 데이터(IndexedDB)는 그대로 유지됩니다.
🔧 문제 해결

데이터가 사라졌어요

백업 파일 → 설정 > 데이터 복원. 백업 없으면 복구 불가.

화면이 이상해요 (업그레이드 후)

브라우저 캐시 문제입니다. 강제 새로고침하거나 캐시를 삭제하세요.

알림이 안 와요

확인방법
알림 활성화설정 > 예약 알림 > 토글 켜기
브라우저 권한브라우저 설정에서 알림 허용
앱 열림 여부앱이 열려있어야 작동

사진 업로드가 안 돼요

파일이 너무 크면 느릴 수 있습니다. JPG, PNG, GIF, WebP를 지원합니다.

❓ FAQ

홈 화면에 추가는 꼭 해야 하나요?

네. 앱처럼 사용 가능하고, Safari 데이터 자동 삭제를 방지합니다.

업그레이드하면 데이터가 날아가나요?

아니요. 코드만 업데이트되고 데이터는 그대로 유지됩니다.

다른 사람도 쓸 수 있나요?

네. URL만 공유하면 됩니다. 데이터는 각 기기에 별도 저장되어 섞이지 않습니다.

알림이 앱을 닫아도 오나요?

아니요. 앱이 열려있을 때만 작동합니다.

PC와 모바일 데이터가 달라요

정상입니다. 각 기기가 독립 데이터입니다. 백업/복원으로 동기화하세요.

30년 후에도 사용 가능한가요?

HTML/JS는 웹 표준이므로 브라우저가 존재하는 한 동작합니다.

비용이 드나요?

완전 무료입니다. 지금도, 앞으로도.
