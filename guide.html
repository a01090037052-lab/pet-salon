A. GitHub Pages 모드: 각자 자신의 기기에서만. PC 서버 모드: 같은 WiFi에서 3~4명 동시 사용 가능.

🔧 문제 해결

데이터가 사라졌어요

백업 파일이 있으면 → 설정 > 데이터 탭 > 데이터 복원하기. 없으면 복구 불가.

화면이 이상해요 (업그레이드 후)

브라우저 캐시 문제입니다. 강제 새로고침 (아래로 당기기 + 새로고침) 또는 캐시 삭제 후 접속.

알림이 안 와요

확인방법
알림 활성화설정 > 알림·메시지 > 예약 알림 토글 ON
브라우저 권한브라우저 설정에서 알림 허용 확인
앱 열림 여부앱이 열려있어야만 작동

사진 업로드가 안 돼요

파일이 너무 크면 느릴 수 있습니다. JPG, PNG, GIF, WebP를 지원합니다. 저장소 용량 경고도 확인하세요.

시간 충돌 경고가 자꾸 떠요

예상 소요시간을 확인하세요. 예: 14:00 시작, 1시간 → 14:00~15:00 점유. 같은 미용사가 이 시간에 다른 예약이 있으면 경고.

❓ FAQ

홈 화면에 추가는 꼭 해야 하나요?

네. 앱처럼 사용 가능하고, Safari 데이터 자동 삭제를 방지합니다.

업그레이드하면 데이터가 날아가나요?

아니요. 코드만 업데이트되고 데이터는 그대로 유지됩니다.

다른 사람도 쓸 수 있나요?

네. URL만 공유하면 됩니다. 데이터는 각 기기에 별도 저장되어 섞이지 않습니다.

알림이 앱을 닫아도 오나요?

아니요. 앱이 열려있을 때만 작동합니다.

PC와 모바일 데이터가 달라요

정상입니다. 각 기기가 독립 데이터입니다. 백업/복원으로 동기화하세요.

포인트와 스탐프의 차이는?

포인트는 금액 기반, 스탐프는 회차 기반. 둘 다 적립/사용 관리 가능.

프로모션은 자동 적용되나요?

네. 마케팅 탭에서 프로모션을 "활성"으로 설정하면 미용 기록 등록 시 자동 적용됩니다.

미수금은 어떻게 관리하나요?

미용 기록에서 "미수금액" 입력 → 대시보드에 누적 → "결제 처리" 버튼으로 정산.

사진 카드는 어디서 만들어요?

미용 기록 저장 후 → "사진 카드 생성" → 레이아웃/테마 선택 → 이미지 다운로드.

30년 후에도 사용 가능한가요?

HTML/JS는 웹 표준이므로 브라우저가 존재하는 한 동작합니다.

비용이 드나요?
