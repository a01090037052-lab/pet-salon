소개

펫살롱이란?

펫살롱은 애견 미용 샵의 전체 운영을 관리하기 위해 설계된 웹 기반 고객 관리 시스템입니다. 매장에서 필요한 모든 기능을 한 곳에서 사용할 수 있습니다.

주요 특징

📱
완벽한 모바일 지원
스마트폰, 태블릿에서 자유롭게 사용
⚡
초고속
로딩 지연 없이 빠른 반응
📴
오프라인 작동
인터넷 없어도 모든 기능 이용 가능
📊
상세한 분석
실시간 매출, 고객 분석 제공
💾
자동 저장
데이터 손실 걱정 없음
📸
사진 저장
미용 전후 사진 카드 생성

시작하기

첫 설정 순서

1
매장 정보 입력

설정 > 매장 관리에서 매장 이름, 전화번호, 주소를 입력합니다.

2
미용사 등록

설정 > 매장 관리에서 담당 미용사 이름을 추가합니다.

3
서비스 메뉴 등록

서비스 메뉴에서 미용 서비스(목욕, 드라이, 트리밍 등)와 가격(소형/중형/대형)을 설정합니다.

4
첫 고객 등록

고객 관리에서 보호자 정보와 반려견 정보를 등록합니다.

💡
팁

설정을 먼저 완료해야 예약 및 기록 작성이 정확하게 작동합니다. 특히 서비스 메뉴 가격 설정이 중요합니다.

일일 사용 가이드

아침: 오늘 예약 확인

앱을 열면 대시보드에 오늘의 예약 목록이 표시됩니다. 몇 건의 예약이 있는지, 어떤 고객들이 올 예정인지 한눈에 확인할 수 있습니다.

대시보드 확인 사항
  • 오늘 예약 건수 및 시간
  • 고객명, 반려견명
  • 미용사별 일정
  • 미수금 현황
  • 재방문 알림 (7일 초과)
  • 생일 축하 알림 (7일 이내)

전화 예약: 고객 및 반려견 검색

  1. 전화를 받으면 좌측 상단의 돋보기 아이콘 또는 /를 눌러 검색창을 엽니다.
  2. 고객 이름이나 반려견 이름, 전화번호를 입력하여 검색합니다.
  3. 기존 고객이면 조회 후 "예약" 버튼을 클릭합니다.
  4. 신규 고객이면 검색 결과 하단의 "+ 새 반려견 등록"을 클릭합니다.
  5. 검색어가 반려견 이름에 자동 입력됩니다.

현장 접수: 신규 고객 빠르게 등록

고객이 방문했을 때 빠르게 등록하려면:

  1. 돋보기 검색 또는 하단 + 버튼 → "새 고객"을 선택합니다.
  2. 반려견 이름(필수)을 먼저 입력합니다.
  3. 연락처(필수)를 입력합니다. 이미 등록된 번호면 실시간 안내가 표시됩니다.
  4. 보호자 이름, 견종, 몸무게는 선택 — 나중에 수정 가능합니다.
  5. 저장 후 바로 예약 등록 또는 상세 보기를 선택합니다.

캘린더에서 빠른 예약

예약 페이지의 캘린더에서 원하는 날짜를 탭하면:

  1. 해당 날짜의 예약 요약(시간 + 반려견 이름)이 바로 표시됩니다.
  2. "+ 이 날짜에 예약 추가" 버튼으로 날짜 입력 없이 바로 예약할 수 있습니다.
  3. 좌우 스와이프로 월을 이동할 수 있습니다.

미용 완료: 기록 작성부터 문자 발송까지

1
미용 기록 작성

예약 목록에서 "미용 완료" 버튼을 클릭하거나, 미용 기록 페이지에서 새 기록을 추가합니다. 서비스, 가격, 다음 예약일 등을 입력합니다.

2
사진 카드 생성

기록 목록에서 📸 버튼을 클릭합니다. 레이아웃(네컷·피드·스토리·2컷·1컷)과 테마(클래식·필름·파스텔·다크·커스텀)를 선택하고 사진을 업로드하면 인스타그램에 최적화된 카드가 생성됩니다. 미리보기 확인 후 공유/고객 전송/다운로드를 선택합니다.

3
다음 예약 제안

고객에게 다음 미용 추천 날짜를 제시하고 예약합니다.

4
완료 문자 발송

미용 완료 안내 문자를 발송합니다. 미용 기록 옆의 "문자" 버튼으로 한 번에 전송할 수 있습니다.

퇴근: 일일 정산

매출 페이지에서:

  • 오늘 총 매출 확인
  • 결제 수단별 통계 확인
  • 미수금 고객 확인 및 연락
  • 필요시 일일 정산표 다운로드
💡
효율 팁

예약 시간대에 미용 기록 완료 여부를 실시간으로 업데이트하면, 매출 추적과 고객 만족도 향상에 도움이 됩니다.

메뉴별 상세 가이드

대시보드

앱을 열었을 때 가장 먼저 보는 페이지입니다. 당일 주요 정보를 한눈에 파악할 수 있습니다.

주요 기능
  • 오늘 예약: 본일 예약 목록 (상태: 대기, 확정, 미용중, 완료, 취소, 노쇼)
  • 오늘 매출: 금일 총 매출 및 건수 (일일 목표 진행률 표시)
  • 미수금 현황: 미지급 건 조회 및 결제 처리
  • 재방문 알림: 마지막 미용 후 설정 기간이 지난 반려견 목록
  • 생일 알림: 7일 내 생일인 고객 및 반려견
  • 백업 알림: 데이터 안전을 위한 백업 권고

예약 관리

고객 예약을 효율적으로 관리하는 페이지입니다.

주요 기능
  • 캘린더 뷰: 월별 캘린더에 예약 바(반려견 이름 + 상태 색상) 표시. 히트맵 배경으로 바쁜 날 한눈에 파악. 좌우 스와이프로 월 이동.
  • 날짜 요약 패널: 캘린더 날짜 탭 → 해당 날짜 예약 목록 + "예약 추가" 버튼 바로 노출
  • 토/일 색상 구분: 토요일(파랑), 일요일(빨강)으로 주말 즉시 구분
  • 오늘 강조: 원형 파란 배경 + "오늘" 버튼으로 빠른 복귀
  • 타임테이블: 미용사별 일정을 시간 단위로 관리 (8~22시)
  • 반복 예약: 주 1회, 2주, 4주 등 정기 예약 생성
  • 예약 상태 변경: 대기 → 확정 → 미용중 → 완료
  • 시간 충돌 감지: 미용사별, 반려견별 중복 예약 방지

미용 기록

완료된 미용 작업을 기록하고 관리합니다.

주요 기능
  • 기록 작성: 서비스, 스타일, 추가항목, 금액, 담당 미용사, 컨디션 기입
  • 날짜 그룹핑: 오늘/어제/이번 주/월별 자동 섹션 구분
  • 필터 합계 바: 검색/필터 결과의 건수·합계·미결제 소계 실시간 표시
  • 사진 카드: 인생네컷·인스타 피드·스토리 등 5종 레이아웃. 4종 테마 + 커스텀(컬러/배경이미지/로고). 미리보기 후 공유/고객전송/다운로드.
  • 영수증 생성: 고객에게 제공할 영수증 자동 생성
  • 결제 상태: 카드, 현금, 이체, 미결제 (칩 선택)
  • 다음 예약일: 미용 주기 자동 계산 + 저장 후 바로 예약
  • 일일 정산표: 결제 수단별·미용사별 집계 + 시간순 상세 내역

고객 관리

고객 정보를 체계적으로 관리합니다.

주요 기능
  • 고객 목록: 이름순, 최근방문순, 등록일순, 방문횟수순 정렬
  • 고객 분류: VIP, 신규, 단골, 주의 태그 설정
  • 고객 상세: 방문 통계, 미용 기록, 반려견 목록 조회
  • 미수금 확인: 고객별 미지급액 추적
  • 전화/문자: 직접 연락 기능
  • 메모: 고객 특이사항 기록

반려견 관리

반려견의 개별 정보와 관리 정보를 기록합니다.

주요 기능
  • 반려견 프로필: 이름, 견종, 체중, 나이, 성별, 중성화 여부
  • 건강 정보: 성격, 건강 특이사항, 알러지, 선호 스타일
  • 사진: 반려견 프로필 사진 업로드
  • 미용 주기: 개별 반려견별 권장 미용 주기 설정
  • 상태 관리: 활동중, 사망, 양도 등 상태 기록
  • 미용 이력: 해당 반려견의 모든 미용 기록 조회

매출 분석

수익성을 분석하고 전략을 수립합니다.

주요 기능
  • 오늘 매출: 본일 매출 목표 진행률
  • 이번 주/달 매출: 기간별 수익 요약
  • 일별/월별 차트: 시각적 매출 추이 분석
  • 결제 수단별 통계: 현금, 카드, 이체 분석
  • 미용사별 매출: 담당자별 성과 파악
  • 손익 계산: 고정비, 변동비 입력 후 순이익 계산
  • CSV 내보내기: 세무사 제출용 자료 추출

서비스 메뉴

제공하는 미용 서비스와 가격을 관리합니다.

주요 기능
  • 서비스 등록: 미용 항목 추가 (목욕, 드라이, 트리밍 등)
  • 사이즈별 가격: 소형, 중형, 대형 동물별로 다른 가격 설정
  • 활성화/비활성화: 서비스 사용 여부 관리
  • 예약에 자동 반영: 설정한 서비스가 예약/기록에서 선택 가능

설정

앱 전체 설정을 관리합니다. 3개 탭으로 구성됩니다.

매장 관리 탭
  • 매장 정보: 이름, 전화번호, 주소, 재방문 알림 기준일
  • 매장 로고: 사진 카드에 자동 삽입되는 로고 이미지 (PNG 투명 배경 권장)
  • 미용사 관리: 담당 미용사 추가/삭제 (1인 살롱은 비워두세요)
  • 휴무일 설정: 정기 휴무일 지정 (캘린더에 표시, 예약 시 경고)
  • 테마 색상: 앱 전체 색상 커스터마이징
운영 설정 탭
  • 알림 설정: 예약 알림 활성화 및 시간 설정
  • 매출 목표: 월 매출 목표 (원 단위)
  • 고정비: 월 고정비 (만원 단위, 손익 계산용)
  • 메시지 템플릿: 7종 SMS 발송 문구 커스터마이징 (변수 자동 치환)
데이터 탭
  • 데이터 현황: 고객/반려견/예약/기록/서비스 수 한눈에 (각 페이지로 이동 가능)
  • 저장 공간 최적화: 사진 데이터를 압축 분리하여 용량 확보
  • 세무 자료 내보내기: 기간별 매출 내역 CSV 파일
  • 백업: 사진 제외 백업(권장, 가벼움) + 사진 포함 백업(용량 주의)
  • 복원: 백업 파일에서 데이터 복원 (자동 안전 백업 포함)
  • 전체 초기화: 모든 데이터 삭제 (2단계 확인)

검색 기능

전체 검색 사용하기

화면 우측 하단의 돋보기 아이콘을 클릭하거나 / 키를 눌러 검색창을 엽니다.

검색 기준

  • 고객 이름: "박민주", "이영희" 등
  • 전화번호 뒷자리: "1234", "9876" 등 4자리
  • 반려견 이름: "뽀삐", "나봉이" 등

검색 결과 활용

  • 검색 결과를 클릭하면 상세 카드를 볼 수 있습니다.
  • 카드에서 전화, 문자, 예약, 미용 기록 등을 빠르게 실행할 수 있습니다.
  • 미수금, 최근 기록, 반려견 정보를 한눈에 확인할 수 있습니다.
💡
검색 팁

전화로 예약을 받을 때 전화번호로 검색하면 기존 고객을 빠르게 찾을 수 있습니다. "9876"만 입력해도 전화번호가 ***-**-9876인 고객을 찾아줍니다.

문자(SMS) 기능

자동 메시지 발송

펫살롱은 4가지 상황에서 자동 문자를 보낼 수 있습니다.

예약 확인 문자

새 예약이 등록되면 고객에게 예약 확인 문자를 보낼 수 있습니다. 아래 정보가 포함됩니다:

  • 예약 날짜 및 시간
  • 반려견 이름
  • 담당 미용사
  • 매장 연락처
미용 완료 안내

미용을 완료하고 기록을 저장할 때 발송합니다:

  • 완료 안내
  • 시술 서비스 항목
  • 금액
  • 감사 인사
재방문 알림

마지막 미용 후 설정된 기간이 지나면 대시보드에 표시되며, "문자" 버튼으로 발송합니다:

  • 마지막 미용 후 경과일
  • 다음 예약 제안
  • 매장 연락처
생일 축하 문자

반려견 또는 고객의 생일 7일 전부터 대시보드에 표시되며, 축하 문자를 발송할 수 있습니다:

  • 생일 축하 인사
  • 특별 할인 제안
  • 매장 연락처

사진 카드 기능

미용 완료 후 인스타그램에 최적화된 사진 카드를 만들어 공유할 수 있습니다.

카드 만들기

  1. 미용 기록 목록에서 📸 버튼을 클릭합니다.
  2. 레이아웃을 선택합니다: 네컷 세로 · 피드(4:5) · 스토리(9:16) · 2컷 · 1컷
  3. 사진을 선택합니다 (레이아웃에 따라 1~4장).
  4. 테마를 선택합니다: 클래식 · 필름 · 파스텔 · 다크 · 🎨커스텀
  5. "카드 생성" → 미리보기 확인 → 공유/고객전송/다운로드

커스텀 테마

"🎨 커스텀"을 선택하면 자유롭게 디자인할 수 있습니다:

  • 빠른 색상 조합: 화이트·베이지·로즈·네이비·민트 프리셋 1탭 적용
  • 직접 선택: 배경색·글자색·포인트색 컬러 피커
  • 배경 이미지: 직접 만든 배경 업로드 (배경색 대신 적용)
  • 텍스트 토글: 반려견 이름·날짜·매장명 개별 표시/숨김

매장 로고

설정 > 매장 관리에서 로고를 등록하면 카드 상단에 자동 삽입됩니다. PNG 투명 배경 권장.

배경 이미지 만들기

  • 권장 크기: 1080×1080 (범용) 또는 1080×1920 (스토리 전용)
  • 만들기: Canva, 미리캔버스에서 배경색/그라데이션 + 테두리 장식
  • 주의: 중앙은 사진 영역, 하단 20%는 텍스트 영역 → 장식은 가장자리에
  • 다운로드: PNG 또는 JPG로 저장 후 앱에서 업로드

인스타그램 최적화

  • 모든 레이아웃은 1080px 이상으로 자동 업스케일 (흐림 방지)
  • 피드: 1080×1350 (4:5) → 피드에서 최대 면적 차지
  • 스토리: 1080×1920 (9:16) → 스토리 전체 화면
  • 생성 후 "공유하기" → 인스타/카톡/에어드롭으로 바로 전송

데이터 관리

백업 방법

두 가지 백업 옵션이 있습니다:

  • 백업 (사진 제외 · 권장): 고객·예약·기록 데이터만 저장. 빠르고 가벼움 (수 MB).
  • 사진 포함 백업: 사진 데이터까지 포함. 용량이 크므로 주의 (수백 MB 가능).
  1. 설정 > 데이터 탭으로 이동합니다.
  2. "백업 (사진 제외 · 권장)" 또는 "사진 포함 백업" 버튼을 클릭합니다.
  3. JSON 파일이 다운로드됩니다. 파일 이름에 날짜+시간이 포함됩니다.
  4. 카톡 나에게 보내기나 클라우드에 보관하면 폰 분실 시에도 안전합니다.
💡 iOS 사용자 주의: iPhone Safari에서는 7일 이상 앱을 사용하지 않으면 데이터가 삭제될 수 있습니다. 홈 화면에 앱을 추가하고, 정기적으로 백업해주세요.
💡
백업 권장사항

최소 1주일에 1번, 데이터가 많을수록 자주 백업하세요. 안전한 클라우드 저장소(Google Drive, iCloud 등)에 복사본을 보관하는 것을 권장합니다.

복원 방법

  1. 설정 > 데이터 탭으로 이동합니다.
  2. "데이터 복원하기" 버튼을 클릭합니다.
  3. 백업 파일(JSON)을 선택합니다.
  4. 확인 메시지를 읽고 진행합니다. (기존 데이터는 덮어쓰기됩니다)
  5. 복원 완료 후 자동으로 이전 데이터를 안전하게 보관합니다.

폰 교체 시 데이터 이전

  1. 기존 폰에서: 설정 > 데이터에서 백업 파일 다운로드
  2. 안전 보관: 백업 파일을 이메일로 전송하거나 USB에 저장
  3. 새 폰에서: 펫살롱 앱 설치 후 복원 기능 사용
  4. 확인: 데이터가 정확히 이전되었는지 확인

백업 파일 보관 중요성

  • 장기 사업 자료 보관: 최소 3년~5년 보관 권장
  • 세무사 상담 시 활용
  • 분쟁 발생 시 증거 자료
  • 새 직원 교육 자료
중요

백업 파일은 매우 중요합니다. 여러 곳에 복사본을 저장하고, 안전한 장소에 보관하세요. 기기 고장이나 분실 시에만 필요하지만, 필요할 때 없으면 모든 데이터를 잃을 수 있습니다.

오프라인 사용법

펫살롱은 오프라인에서도 작동합니다

펫살롱은 PWA(Progressive Web App)로 만들어져서, 한 번 접속하면 인터넷 없이도 사용할 수 있습니다.

오프라인 동작 원리
  • 앱 파일: 처음 접속 시 모든 파일이 기기에 자동 저장(캐시)됩니다
  • 고객 데이터: 모든 데이터는 기기 내부(IndexedDB)에 저장됩니다. 서버로 전송되지 않습니다
  • 인터넷 불필요: 고객 등록, 예약, 미용 기록, 매출 확인 등 모든 기능이 오프라인에서 동작합니다

홈 화면에 앱 설치하기 (추천)

홈 화면에 설치하면 일반 앱처럼 사용할 수 있고, 오프라인 안정성도 높아집니다.

아이폰 (Safari)

  1. Safari로 펫살롱 접속
  2. 하단의 공유 버튼 (네모+화살표) 탭
  3. "홈 화면에 추가" 선택
  4. "추가" 탭 → 홈 화면에 펫살롱 아이콘 생성

안드로이드 (Chrome)

  1. Chrome으로 펫살롱 접속
  2. 주소창 아래 "홈 화면에 추가" 배너 또는 우측 상단 ⋮ > 홈 화면에 추가
  3. "설치" 탭 → 홈 화면에 앱 아이콘 생성

문제 해결 (FAQ)

Q. 앱이 안 열려요

앱을 여러 번 사용했는데 갑자기 안 열리는 경우가 있습니다. 다음을 시도해보세요: 1) 브라우저 새로고침 (F5 또는 Ctrl+R), 2) 브라우저 캐시 삭제 후 다시 접속, 3) 다른 브라우저 사용, 4) 디바이스 재시작

Q. 데이터가 사라졌어요

먼저 당황하지 마세요! 펫살롱은 기기에 데이터를 저장하므로 웹사이트를 방문하는 방식에 따라 달라질 수 있습니다. 1) 북마크된 정확한 주소로 접속했는지 확인, 2) 홈 화면에 추가했으면 그 아이콘으로 시작, 3) 백업 파일이 있으면 복원 기능으로 데이터 복구

Q. 오프라인에서 사용하려면?

펫살롱은 완벽한 오프라인 지원을 제공합니다. 1) 한 번 접속하면 필요한 파일이 자동 다운로드되고, 2) 이후 인터넷 없이도 모든 기능 사용 가능, 3) 연결 복구 후 자동 동기화. 인터넷 없는 상황에서도 걱정 없이 사용하세요.

Q. 문자가 안 보내져요

1) 고객 연락처가 정확히 입력되었는지 확인, 2) 휴대폰의 SMS 권한이 허용되었는지 확인, 3) 앱 개발자의 SMS 기능은 문자 작성만 자동화하며, 실제 발송은 기기의 기본 문자 앱을 거치므로 발송 여부는 기기에 따라 다를 수 있음

Q. 예약이 겹친다고 나와요

펫살롱은 같은 미용사나 같은 반려견의 예약 시간이 겹치는 것을 방지합니다. 1) 예약 소요시간 재확인, 2) 다른 미용사 배정 또는 시간 변경, 3) 기존 예약 상태 변경 확인

Q. 저장 공간이 부족하다고 해요

사진이 많으면 저장 공간을 많이 사용합니다. 1) 오래된 사진 삭제, 2) 백업 파일 다운로드 후 기기에서 삭제, 3) 별도 클라우드 저장소 활용 고려

Q. 모바일과 PC에서 동시에 쓸 수 있나요?

각 기기는 독립적인 데이터를 가집니다. 데이터를 동기화하려면: 1) 한 기기에서 백업 파일 생성, 2) 다른 기기에서 복원 기능으로 불러오기. 이 방법으로 여러 기기에서 같은 데이터를 사용할 수 있습니다.

Q. 비용이 드나요?

펫살롱은 완전 무료입니다. 광고도 없고, 추가 구독료도 없습니다. 다만 매장의 인터넷 요금은 발생합니다(초기 설정 시에만 필요).

유지보수 불가 시 대처법

현재 상황 이해

펫살롱은 GitHub Pages라는 무료 호스팅으로 운영되고 있습니다. 이는 개발자가 관리하지 못하는 상황이 생길 수 있음을 의미합니다. 다음은 대비책입니다.

1단계: 정기 백업

  • 최소 주 1회 백업 수행 (설정 > 데이터 > 데이터 백업하기)
  • 여러 곳에 복사본 보관: 외장하드, 클라우드, 이메일
  • 중요 데이터(고객 목록, 재방문 주기)는 엑셀로도 별도 관리 고려

2단계: 다른 호스팅 이용

개발자가 미지우면 다른 호스팅 서비스로 옮길 수 있습니다:

  • Vercel, Netlify 등 무료 호스팅 서비스 이용 가능
  • 자신의 서버/웹호스팅에 설치 가능
  • IT 기술자의 도움 필요 (완전히 자동화되지 않음)

3단계: 다른 개발자에게 인수

기술 능력이 없다면:

  • 개발자 커뮤니티에 요청
  • 백업 파일 제공으로 새로운 버전 개발 요청
  • 유사한 기능의 다른 솔루션 찾기

4단계: 다른 시스템으로 데이터 이전

만약 완전히 다른 시스템으로 옮겨야 한다면:

  1. 펫살롱에서 백업 파일 다운로드
  2. 고객 목록, 반려견 정보, 예약, 기록을 CSV 또는 엑셀로 변환
  3. 새로운 CRM 시스템 또는 미용 관리 소프트웨어로 가져오기
💡
미리 준비하기

사업이 중요할수록 백업이 중요합니다. 정기적인 백업은 어떤 상황에서든 데이터 손실을 방지합니다. 오늘부터 시작하세요!

최악의 시나리오: 데이터 복구

혹시 백업 없이 데이터를 잃어버렸다면:

  • 기기의 브라우저 캐시/쿠키에 일부 복구 가능할 수 있습니다
  • IT 전문가에게 기기의 IndexedDB 데이터 복구 문의 가능
  • 이를 위해서도 정기 백업이 최고의 방법입니다
매우 중요

무료 도구를 사용할 때의 위험성을 항상 인식하세요. 사업의 핵심 데이터는 반드시 정기적으로 백업하고, 여러 위치에 보관해야 합니다.

엑셀/노션 연동 가이드

CSV 파일을 엑셀에서 열기

펫살롱에서 내보낸 CSV 파일을 엑셀에서 바로 활용할 수 있습니다.

  1. CSV 다운로드: 매출 페이지 또는 미용 기록 > 세무 자료 내보내기 > 기간 선택 > CSV 다운로드
  2. 엑셀에서 열기: 다운로드된 CSV 파일을 더블클릭하면 엑셀에서 자동으로 열립니다
  3. 한글 깨짐 시: 엑셀 > 데이터 > 텍스트/CSV 가져오기 > 파일 선택 > 인코딩 "UTF-8" 확인 후 가져오기

엑셀에서 매출 분석하기

CSV를 열면 다음 열이 포함됩니다:

날짜, 고객명, 반려견, 견종, 서비스, 기본금액, 할인, 추가요금, 최종금액, 결제수단, 담당 미용사, 메모

피벗 테이블로 분석 (엑셀)

  1. 데이터 영역 선택 후 삽입 > 피벗 테이블
  2. 견종별 매출: 행에 "견종", 값에 "최종금액"(합계)
  3. 월별 매출: 행에 "날짜"(월 그룹화), 값에 "최종금액"(합계)
  4. 미용사별 실적: 행에 "담당 미용사", 값에 "최종금액"(합계) + "최종금액"(개수)
  5. 서비스 인기도: 행에 "서비스", 값에 "최종금액"(개수)
💡
구글 스프레드시트 사용 시

구글 드라이브에 CSV를 업로드하면 구글 스프레드시트에서 바로 열 수 있습니다. 피벗 테이블도 동일하게 사용 가능합니다.

노션(Notion)에서 활용하기

노션에 매출 데이터를 가져와서 데이터베이스로 관리할 수 있습니다.

CSV를 노션으로 가져오기

  1. 노션에서 새 페이지 생성
  2. /table 입력 > "Table - Full page" 선택
  3. 우측 상단 ... 메뉴 > Merge with CSV (CSV 병합)
  4. 다운로드한 CSV 파일 선택
  5. 데이터가 테이블로 자동 생성됩니다

노션 데이터베이스 활용 팁

  • 필터: 견종별, 결제수단별, 미용사별로 필터링
  • 정렬: 날짜순, 금액순 정렬
  • 뷰 전환: 테이블, 보드, 캘린더, 갤러리 등 다양한 뷰 활용
  • 수식: 노션 수식으로 월별 합계, 평균 단가 계산 가능

추천 워크플로우

  1. 매월 1일에 지난달 CSV 내보내기
  2. 노션 "매출 기록" 데이터베이스에 CSV 가져오기
  3. 노션에서 월별 요약 페이지 작성
💡
노션 무료 플랜 주의

노션 무료 플랜에서도 CSV 가져오기와 데이터베이스 기능을 사용할 수 있습니다. 다만 파일 업로드 용량에 제한이 있으니, 너무 오래된 데이터는 분기별로 나눠서 관리하세요.
